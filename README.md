# 🎵 Downtube - YouTube Downloader

Aplicação desktop desenvolvida em **Python** que permite baixar vídeos e músicas do YouTube de forma simples e rápida através de uma interface gráfica intuitiva.

<img width="1605" height="612" alt="image" src="https://github.com/user-attachments/assets/a7b2744b-c47c-4de7-92a1-5c07b33ee9ab" />


## 📋 Sobre o Projeto

O Downtube foi criado para facilitar o download de conteúdos do YouTube, oferecendo opções para baixar:

* 🎵 Apenas o áudio (MP3)
* 🎥 Vídeo em alta qualidade (MP4 High)
* 🎥 Vídeo em baixa qualidade (MP4 Low)

O projeto utiliza Tkinter para a interface gráfica, Pytube para realizar os downloads e TTKBootstrap para um visual moderno.

---

## 🚀 Funcionalidades

✅ Download de áudio do YouTube

✅ Download de vídeos em alta resolução

✅ Download de vídeos em baixa resolução

✅ Seleção da pasta de destino

✅ Interface gráfica simples e intuitiva

✅ Tema visual moderno com TTKBootstrap

## 📦 Instalação

### Clone o repositório

```bash
git clone https://github.com/Gabsxzin/Downtube.git
```

### Acesse a pasta

```bash
cd Downtube
```

### Instale as dependências

```bash
pip install pytube ttkbootstrap
```

---

## ▶️ Como Executar

Execute o arquivo principal:

```bash
python downtube.py
```

---

## 🎮 Como Utilizar

1. Copie o link do vídeo do YouTube.
2. Cole o link no campo de entrada.
3. Escolha o formato desejado:

   * MP3
   * MP4-HIGH
   * MP4-LOW
4. Clique em **Download**.
5. Selecione a pasta onde deseja salvar.
6. Aguarde a conclusão do download.

---

## 📸 Interface

A aplicação possui:

* Campo para inserir o link do vídeo
* Seletor de formato
* Botão de download
* Seleção da pasta de destino

---

## ⚠️ Observações

* É necessário possuir conexão com a internet.
* Alguns vídeos podem não estar disponíveis para download devido a restrições do YouTube.
* Dependendo da versão do YouTube, a biblioteca Pytube pode exigir atualizações.
